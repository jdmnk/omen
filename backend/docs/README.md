# Backend API Notes

Reference notes for upstream Polymarket APIs and hosted Goldsky subgraph queries used by the backend.

- `POLY_GENERAL.md` - Gamma API event and market discovery notes.
- `SEARCH_API.md` - Gamma public search request/response shape.
- `BOOKS_API.md` - CLOB order book summaries.
- `PRICES_API.md` - CLOB price lookup.
- `GRAPHS.md` - hosted Goldsky wallet and PnL subgraph query examples.

These files are implementation notes, not generated API docs. For the local Omen API, start the backend and open `http://localhost:8000/docs`.
